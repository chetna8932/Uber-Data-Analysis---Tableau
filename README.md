# 🚖 NCR Ride-Hailing Analytics Dashboard

An interactive Tableau dashboard analyzing ride-hailing data across Delhi NCR — covering bookings, cancellations, ratings, revenue, and more.

**Live Dashboard:** [View on Tableau Public](#)

---

## 📁 Dataset — `NCR.xlsx`

Key fields: Booking ID, Status, Vehicle Type, Pickup/Drop Location, Booking Value, Ride Distance, VTAT, CTAT, Driver & Customer Ratings, Payment Method, Cancellation Reasons.

---

## 📈 Key Insights

- Booking status breakdown (Completed / Cancelled / Incomplete / No Driver Found)
- Vehicle type popularity & revenue trends
- Cancellation reasons by driver and customer
- Arrival time (VTAT & CTAT) efficiency
- Ratings and payment method distribution
- Top pickup & drop locations across NCR
